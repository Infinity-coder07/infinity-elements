# 🧪 Elements Search

A fast and lightweight web app to explore detailed information about chemical elements.

---

## 🚀 Features

- 🔍 Search using:
  - Atomic Number  
  - Element Name  
  - Symbol  
  - Atomic Mass  

- ⚡ Instant results  
- 📱 Mobile-friendly  
- 💾 Works offline (no API needed)

## 📊 Information Provided

- Atomic Number  
- Element Name  
- Symbol  
- Atomic Mass (amu)  
- Atomic Mass (Without Decimal)  
- Type  
- Group  
- Period  
- Block  
- State at STP  
- Color  
- Electron Configuration  
- Electronegativity (Pauling)  
- Electron Affinity (eV)  
- Atomic Radius (van der Waals, pm)  
- First Ionization Enthalpy (eV)  
- Density (g/cm³)  
- Melting Point (K)  
- Boiling Point (K)  
- Specific Heat (J/g·K)  
- Common Oxidation States  
- Discoverer  
- Year of Discovery  
- Common Uses / Occurrences  
- Special Characteristics
